## Descrição:

- Esse app tem como objetivo utilizar o LogCat para exibir informações coletadas dos campos de texto preenchidos, se não estiverem preenchidos, ficarão bloqueados até inserir ao menos um caractér.

### Exemplos:

- ``$nome, avaliou o filme, $filme, como ótimo``
- ``$nome, avaliou o filme, $filme, como bom``
- ``$nome, avaliou o filme, $filme, como razoável``
- ``$nome, avaliou o filme, $filme, como ruim``
